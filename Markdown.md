## Списки

Чтобы добавить ненумерованные списки, необходимл пункты выделить (*). Например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например:
1. Пункт
2. Пункт
3. Пункт
